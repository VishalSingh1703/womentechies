Key Features

FILTERING THE MAILS

Our app filter out the mails according to the given keywords, which can be added easily by user.

ORGANIZING IMPORTANT MAILS

The app uses labels/colors to auto save email in different categories so that you can not miss any important event and can save your time to read useless spam mails

event scheduler

The app auto updates the calendar for important events accessed from emails and reminds on events and meets regularly through notifications
